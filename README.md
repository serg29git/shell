RU

установка:
1. если shell-installer.sh говорит "Permission Denied' когда его врубаешь:
chmod +x /там/где/shell/штука/*

2. первая штука в shell-installer которая появляется после скачивания (test shell) может не сработать :
перезагрузите терминал. если нифига, еще раз запустите shell-installer. еслми ВСЕ ЕЩЕ нифига, пишите в issues.

3. если не открывается installer в файловом менеджере (dolphin например):

иногда installer может нифига никакой терминал не открываать из менеджера, а просто висеть и ниче не делать.
попробуйте запустить прямо из терминала. если все еще нифига:
chmod +x /там/где/shell/штука/*
